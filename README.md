# Walksies

A small browser prototype about a dog and owner fighting the leash while chasing different goals.

## Run
- Open [index.html](/Users/premi/Desktop/walksies/index.html) directly in a browser, or
- From this folder run `python3 -m http.server 8000` and visit `http://localhost:8000`

## Controls
- Dog solo: arrow keys
- Owner solo: arrow keys
- Two-player:
  - Owner: arrow keys
  - Dog: `WASD`
- `R`: restart current round
- `P`: pause

## Goal
- Reach the glowing home marker on the right side of the map.
- In solo modes, do not let the AI counterpart's anger meter fill up.
- In two-player, the round ends at home and higher score wins.
